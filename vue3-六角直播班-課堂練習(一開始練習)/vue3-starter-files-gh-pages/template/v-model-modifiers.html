v-model 修飾符

檔名：v-model-modifiers.html

修飾符

延遲 Lazy

在 v-model +上 v-model.lazy，
可以讓離開當下輸入框時才呈現值 {{ lazyMsg }}

純數值 Number

如果希望 input 輸入框只能塡數字，那要將 type="number"，
確保輸入的內容是數字，要在 input 輸入框加上 v-model.number，
以防可能都是 string 型態
{{ numberMsg }}{{ typeof numberMsg }}

修剪 Trim

v-model.trim 可以修剪掉 輸入框內 前後空白
這是一段{{ trimMsg }}緊黏的文字
